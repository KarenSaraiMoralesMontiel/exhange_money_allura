# Exchange Rate Console App

Aplicación de consola que consume la API  **Exchange Rate API**.

La aplicación permite 6 conversiones :

1) Dólar =>> Peso argentino
2) Peso argentino =>> Dólar
3) Dólar =>> Real brasileño
4) Real brasileño =>> Dólar
5) Dólar =>> Peso colombiano
6) Peso colombiano =>> Dólar

También guarda todas las conversaciones realizadas y te permite verlas.
